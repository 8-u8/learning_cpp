# ここでのモチベーション
- 主にアルゴリズム・数理最適化の領域のキャッチアップ。
  - データサイエンス以外、というかデータサイエンスを支える技術群の理解に務めるモチベーション
  - ゲーム理論の均衡問題を解くヒントが欲しい。
- ひいては「プログラミングそのもの」に対する理解の深化に務める。
